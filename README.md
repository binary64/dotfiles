# My dotfiles

Feel free to roast, install, or skip.

## Quickstart

```bash
nix-shell -p git --run "git clone https://github.com/binary64/dotfiles ~/dotfiles"
echo -n "123" >/tmp/cryptroot.key
cd ~/dotfiles && bash ./install.sh
sudo reboot now
```

## Features

- Home Manager with Flakes
- tmux
